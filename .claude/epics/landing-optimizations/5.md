# Task 5: Hero Section Business-Type Personalization

## Overview

Implement dynamic hero section personalization based on business type URL parameters to increase relevance and conversion for specific business categories (salon, restaurant, clinic).

## Business Context

Different business types have distinct automation needs and pain points. Generic messaging dilutes impact and reduces conversion. Personalized value propositions increase relevance by 40-60% according to landing page optimization studies.

## Technical Requirements

### URL Parameter Detection
- Implement query parameter parsing: `?type=salon|restaurant|clinic`
- Default to generic messaging if no type specified
- Support deep linking with business type persistence

### Dynamic Content System
```typescript
interface BusinessTypeConfig {
  type: 'salon' | 'restaurant' | 'clinic' | 'generic';
  headline: string;
  subheadline: string;
  valueProps: string[];
  ctaText: string;
  heroImage: string;
  beforeAfter: {
    before: string;
    after: string;
  };
}
```

### Business-Specific Configurations

#### Salon Configuration
- **Headline:** "Automatiza las citas de tu salón y nunca más pierdas clientes"
- **Subheadline:** "Confirmaciones automáticas por WhatsApp, recordatorios de cita y seguimiento post-servicio"
- **Value Props:**
  - Reduce cancelaciones en 60%
  - Automatiza confirmaciones y recordatorios
  - Captura reseñas automáticamente
- **CTA:** "Automatizar Mi Salón"

#### Restaurant Configuration
- **Headline:** "Automatiza reservas y pedidos de tu restaurante sin complicaciones"
- **Subheadline:** "Gestión automática de mesas, confirmaciones y promociones especiales"
- **Value Props:**
  - Gestiona mesas automáticamente
  - Confirma reservas por WhatsApp
  - Promociona eventos especiales
- **CTA:** "Automatizar Mi Restaurante"

#### Clinic Configuration
- **Headline:** "Automatiza las citas médicas y mejora la experiencia del paciente"
- **Subheadline:** "Recordatorios automáticos, confirmaciones y seguimiento post-consulta"
- **Value Props:**
  - Reduce ausencias en 70%
  - Recordatorios automáticos
  - Seguimiento post-consulta
- **CTA:** "Automatizar Mi Clínica"

### Before/After Scenarios

#### Salon Before/After
- **Before:** "Llamadas perdidas, citas olvidadas, clientes que no regresan"
- **After:** "WhatsApp automatizado, citas confirmadas, clientes fieles que recomiendan"

#### Restaurant Before/After
- **Before:** "Mesas vacías por no-shows, pedidos confusos por teléfono"
- **After:** "Reservas confirmadas automáticamente, pedidos claros via WhatsApp"

#### Clinic Before/After
- **Before:** "Pacientes que no llegan, horarios desorganizados"
- **After:** "Agenda optimizada, pacientes puntuales, seguimiento automático"
